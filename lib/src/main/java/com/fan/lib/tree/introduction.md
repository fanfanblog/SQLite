# 普通二叉树伪代码实现
抽象数据类型：节点
   包含本身信息item
   包含左子节点信息leftNode
   包含右子节点信息rightNode
   包含父节点信息parentNode
添加一个节点代码流程：
   1,newNode.compareto(root)
   2,如果小于root,则循环遍历root的左子树，进行插入
   3，如果大于root，则循环遍历root的右子树，进行插入
